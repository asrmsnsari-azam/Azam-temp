Project Overview

This Student Registration Website is a complete web application developed as a final project by Azam Iqbal for the Web Technologies and Database Administration & Management (DBA&M) Lab course. It demonstrates the integration of frontend technologies (HTML, CSS, JavaScript) with backend database connectivity using PHP and MySQL.

Features Implemented

  • Responsive Design: Built with Bootstrap 5 for mobile-first, responsive layout
  • Media Integration: Images, audio, and video elements with controls
  • Form Validation: Client-side JavaScript validation for all fields
  • File Upload: Picture upload with type and size validation
  • Database Storage: Secure data storage in MySQL with password hashing
  • Data Display: View all registered students in a formatted table
